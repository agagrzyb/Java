Multiple JS Files

This HTML file loads three different JavaScript files. Note the load order within the <head> element:

  1. mcr_sf_0.1.js - this is going to be our mycourseresource (mcr) JavaScript file with standard function (sf).
  2. mcr_sp_0.1.js - this is going to be our mycourseresource (mcr) JavaScript file with our something pretty (sp) function. These functions may require the mcr_sf_0.1.js file.
  3. script.js - This is the bespoke file specific for the current website project. Functions in this file may avail of functions from the mcr_sf_0.1.js and mcr_sp_0.1.js files.
